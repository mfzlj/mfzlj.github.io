04. Introduction to TensorFlow

Introduction to TensorFlow

UD230 L1 Intro To Neural Networks With Tensor Flow

TensorFlow is an open-source library developed by Google to help you create and train machine learning models. The fundamental data structure for neural networks are known as tensors, and TensorFlow is built around these tensors.

Tensors

Which of the following arrays qualify as tensors?

SOLUTION:
  • 1-dimensional array
  • 2-dimensional array
  • An array with 3 or more dimensions
  • n-dimensional array
