08. MNIST

MNIST

UD230 L5 MNIST Dataset

There are plenty of different datasets available from the tensorflow_datasets library, which we shortened in the code to tfds . Loading one of the datasets is simple with the tfds.load() function, which takes in the dataset name (in this case 'mnist' ), as well as some other optional arguments such as: 1) the dataset split to get (training, test, validation), 2) whether to shuffle the data, 3) if the data is to be used as part of a supervised learning algorithm (including labels), 4) whether to include metadata about the dataset itself, and more .

You can use the .take() function with an integer as an argument to get a certain number of images at once from the dataset.
